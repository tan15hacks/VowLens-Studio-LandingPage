# VowLens Studio Landing Page

A premium wedding and event photography landing page concept created as a Fiverr portfolio project.

## Brand concept

VowLens Studio is a modern wedding and event photography brand capturing timeless love stories through elegant, emotional, and cinematic visuals.

## Features

- Cinematic hero section
- Premium sticky navigation
- Mobile menu
- Featured gallery with image preview interaction
- Services section
- Wedding package cards
- Testimonials
- Booking process section
- FAQ accordion
- Inquiry form validation
- Toast notification
- SEO metadata
- Open Graph social preview
- Sitemap and robots setup
- Custom loading screen
- Custom 404 page

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide React
- Vercel-ready deployment

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
